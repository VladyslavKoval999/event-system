# Творчі інтенсиви (Лабораторна робота №3)

Проєкт для агрегації даних та візуалізації аналітики реєстрацій на творчі курси. Виконано на Просунутий рівень.

## Технології
* **React 18 + Vite** — для швидкої збірки та розробки.
* **Redux Toolkit** — для управління глобальним станом (збереження улюблених курсів).
* **Recharts** — для побудови адаптивних SVG-графіків.
* **Axios** — для імпорту сутностей із зовнішнього відкритого API.
* **React Router** — для навігації (SPA).
* **Zod + React Hook Form** — для валідації даних форми.
